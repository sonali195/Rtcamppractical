A plugin for creating a slideshow from uploaded images.

== Description ==

This plugin allows you to create a slideshow from images uploaded via the admin settings page. Use the [myslideshow] shortcode to display the slideshow on your site.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/rtcmp-slideshow-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->My Slideshow Plugin screen to configure the plugin.

== Frequently Asked Questions ==

= How do I add images to the slideshow? =

You can add images to the slideshow from the My Slideshow Plugin settings page in the WordPress admin.
